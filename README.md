Elattes
--------

Requirements
------------

- Tools required:
  - [R](https://www.r-project.org/)
  - [R-Studio](https://www.rstudio.com/)

Installation
------------

```bash
install.packages("devtools") 
library(devtools)
devtools::install_github("brunojus/elattes", dependencies =T)
```
